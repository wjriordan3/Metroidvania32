# Metroidvania32
Theme: Defective Side Effects

Concept: Swappable limbs for spare parts

Premise: Mecha Mercenaries, ready for a big height, use their raggedy mecha gear to pull off a smash and grab! Locate all the keys to the vault and fend off the Armored Police!

Art Style: 90s Retro Mecha Anime (Gundam, Gunbuster) 

Mechanic Ideas:
	- Swappable parts for player
  - Synthesize/Desynthesize parts
  - Hook shot? Attaching and detaching

Enemies:
  - Bobots
  - Armored police
